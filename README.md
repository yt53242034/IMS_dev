# ProIMS 原彩專業庫存管理系統 (Professional Inventory Management System)

ProIMS 是一個專為**平板觸控操作**與**桌面環境**設計的現代化進銷存管理系統。採用 React 生態系構建，具備**離線優先 (Offline-First)** 的資料儲存架構，無需架設後端伺服器即可在單機流暢運行。

本系統特別針對中小企業、倉儲管理或個人工作室設計，整合了物料管理、訂單處理、客戶資料庫與強大的備份還原功能。

---

## ✨ 核心功能特色 (Key Features)

### 📊 智慧儀表板 (Smart Dashboard)
*   **即時數據監控**：首頁即時顯示總庫存品項、低庫存警示數、以及近期異動筆數。
*   **一鍵快篩**：
    *   頂部提供分類快篩按鈕。
    *   專屬「缺貨 (Out of Stock)」與「低庫存」篩選器。
*   **全域搜尋**：支援品名、編號(ID)、規格、材質的多欄位模糊搜尋。

### 📦 進階庫存管理 (Inventory Management)
*   **詳細物料資訊**：
    *   基本資料：品名、自訂編號 (支援自動生成)、分類、規格、材質。
    *   庫存控制：現有庫存、安全庫存水位、單價、儲位、供應商。
*   **圖片管理系統**：
    *   支援物料圖片上傳。
    *   **高效能儲存**：圖片自動儲存於裝置內部 (IndexedDB/File System)，非單純 Base64 字串，確保大量圖片時系統依然流暢。
    *   列表縮圖預覽與點擊放大檢視。
*   **Excel 整合**：
    *   **批次匯入**：支援 `.xlsx`, `.xls`, `.csv` 格式，自動對應欄位 (品名, 分類, 庫存...) 並建立資料。
    *   **報表匯出**：可將當前篩選結果匯出為 CSV 報表。
*   **報表列印 (PDF)**：
    *   內建列印功能，可將當前庫存清單轉換為適合列印的格式或另存為 PDF 盤點單。
*   **快速操作**：列表頁面即可直接增減庫存，支援管理員權限控管。

### 🔄 批次作業與訂單系統 (Batch & Orders)
*   **批次出入庫**：
    *   支援「購物車模式」選取多樣物料。
    *   一次性執行大量入庫或出庫作業。
*   **訂單管理**：
    *   建立「進貨單」與「出貨單」。
    *   **暫存機制**：訂單可先建立為「待處理 (Pending)」，確認後再「執行 (Execute)」扣除庫存。
    *   訂單歷史紀錄查詢與刪除。

### 🛡️ 資料安全與備份 (Data Security & Backup)
*   **完整系統備份 (ZIP)**：
    *   使用 `JSZip` 技術，將所有文字資料 (JSON) 與**所有物料圖片**打包成單一 `.zip` 壓縮檔。
    *   解決了傳統 JSON 備份無法有效處理大量圖片的問題。
*   **彈性還原**：
    *   支援匯入 ZIP 備份檔 (含圖片) 與舊版 JSON 備份檔。
    *   還原前可預覽備份資訊 (日期、備註)。
    *   支援**選擇性還原**：僅還原庫存、僅還原訂單、或還原系統設定。
*   **異動歷程 (Audit Log)**：
    *   系統自動記錄每一筆庫存變動 (來源：手動調整、訂單執行、批次作業、Excel 匯入)。
    *   可隨時查閱操作時間、變動數量與相關備註。

### ⚙️ 客製化與設定 (Customization)
*   **多樣化主題**：內建多款配色 (工程藍、深空灰、暖陽紙等)，支援深色模式 (Dark Mode)。
*   **系統規則**：
    *   **允許負庫存**：可設定是否允許庫存扣至負數。
    *   **顯示縮圖**：可開關列表圖片顯示以節省版面。

## 🛠️ 技術架構 (Tech Stack)

本專案採用現代化前端技術堆疊，強調效能與使用者體驗：

*   **核心框架**: React 18 + Vite (極速開發與建置)
*   **UI 樣式**: Tailwind CSS (Utility-first CSS 框架)
*   **動畫效果**: Framer Motion (流暢的轉場與互動動畫)
*   **圖示庫**: Lucide React (輕量且一致的圖示集)
*   **資料處理**:
    *   `xlsx`: 處理 Excel 檔案讀寫。
    *   `jszip`: 處理資料與圖片的壓縮備份。
    *   `react-hot-toast`: 優雅的通知提示。
*   **資料持久化**:
    *   `LocalStorage`: 儲存 JSON 文字資料 (庫存、訂單、設定)。
    *   `IndexedDB` / `File System`: 儲存二進位圖片資料 (透過自定義 `imageHandler` 模組)。

## 🚀 安裝與開發指南

### 前置需求
*   Node.js (建議 v16 以上)

### 1. 下載與安裝
```bash
# 複製專案
git clone [repository-url]
cd proims-app

# 安裝依賴套件
npm install
```

### 2. 啟動開發環境
```bash
npm run dev
```
*   預設開啟於: `http://localhost:5173`

### 3. 平板/手機實機測試
若要在同一 Wi-Fi 下的平板測試，請使用 host 模式啟動：
```bash
npm run dev -- --host
```
*   終端機會顯示 Network URL (例如 `http://192.168.1.100:5173`)，在平板瀏覽器輸入該網址即可。

### 4. 建置生產版本
```bash
npm run build
```
建置後的檔案位於 `dist` 資料夾，可部署至任何靜態網站託管服務 (如 Vercel, Netlify, GitHub Pages)。

## 📂 專案結構簡介

```text
src/
├── components/         # UI 組件 (Modals, Managers, etc.)
│   ├── modals/         # 各式彈出視窗 (新增物料, 設定, 歷史紀錄...)
│   ├── ClientManager   # 客戶管理模組
│   └── OrderManager    # 訂單管理模組
├── contexts/           # React Context (ThemeContext)
├── data/               # 初始測試資料
├── utils/              # 工具函式庫
│   ├── fileHelpers.js  # 檔案下載、CSV 匯出、列印邏輯
│   ├── imageHandler.js # 圖片儲存與讀取邏輯 (IndexedDB)
│   └── idGenerator.js  # ID 自動生成邏輯
└── App.jsx             # 主應用程式邏輯
```
