# 原彩IMS系統 (ProIMS) - 專業庫存管理系統

這是一個基於 React 和 Tailwind CSS 開發的現代化庫存管理系統，專為平板與桌面環境設計。具備完整的進銷存功能、離線資料儲存 (Local Storage)、以及友善的觸控介面。

## ✨ 主要功能

### 📊 儀表板 (Dashboard)
*   **即時概況**：檢視總品項數、低庫存警示、缺貨狀況。
*   **快速存取**：一鍵篩選低庫存/缺貨物料，查看近期異動紀錄。
*   **視覺化數據**：直觀的數據卡片設計。

### 📦 庫存管理 (Inventory)
*   **完整資料欄位**：品名、編號(ID)、規格、材質、分類、庫存量、安全庫存、單價、儲位等。
*   **Excel 整合**：
    *   **匯入**：支援 `.xlsx`, `.xls`, `.csv` 檔案批次匯入物料資料。
    *   **匯出**：可將庫存清單匯出為 Excel (.csv)。
*   **圖片支援**：支援圖片上傳 (自動壓縮) 與預覽，整合 Capacitor 檔案系統以優化效能。
*   **快速操作**：列表直接調整庫存、搜尋與多條件篩選功能。

### 🔄 進出庫作業 (Transactions)
*   **批次處理**：支援一次選取多項物料進行入庫或出庫，提升作業效率。
*   **防呆機制**：支援「允許負庫存」開關設定，避免操作失誤。
*   **異動紀錄**：自動記錄所有庫存變動，包含操作時間、類型、對象與備註。

### 📝 訂單與客戶管理
*   **訂單系統**：建立進貨/出貨訂單，支援暫存、執行與歷史查詢。
*   **客戶資料庫**：管理客戶與供應商資訊 (統編、聯絡人、電話、地址)。

### 📈 報表與備份
*   **PDF 列印**：內建 PDF 產生器，可列印庫存盤點報表與異動明細。
*   **資料備份**：完整系統資料匯出 (JSON)。
*   **彈性還原**：支援選擇性還原 (僅還原庫存、訂單、或設定等)。

### ⚙️ 個人化設定
*   **主題切換**：內建多款深色/淺色主題 (Dark/Light Mode)，如工程藍、暖陽紙、深空灰等。
*   **系統規則**：可設定是否顯示縮圖、是否允許負庫存。

## 🛠️ 技術棧 (Tech Stack)

*   **Frontend Framework**: React.js (Vite)
*   **Styling**: Tailwind CSS
*   **Icons**: Lucide React
*   **Data Handling**: SheetJS (xlsx) for Excel import/export
*   **Mobile/Native Integration**: Capacitor (Filesystem API for Android storage)
*   **PDF Generation**: html2pdf.js

## 🚀 安裝與執行

確保您的環境已安裝 Node.js。

1.  **安裝依賴套件**:
    ```bash
    npm install
    ```

2.  **啟動開發伺服器**:
    ```bash
    npm run dev
    ```

3.  **建置生產版本**:
    ```bash
    npm run build
    ```

## 📱 行動裝置支援

本專案針對平板觸控操作優化：
*   **觸控友善**：加大按鈕尺寸，優化點擊區域。
*   **視覺回饋**：針對觸控裝置調整按鈕的 Active 狀態，移除不必要的 Hover 效果。
*   **原生整合**：透過 Capacitor 支援 Android 原生打包，並整合檔案系統權限。
