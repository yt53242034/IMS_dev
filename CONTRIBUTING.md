# 貢獻指南 (Contributing Guide)

感謝您有興趣參與 ProIMS 的開發！這份文件將協助您快速上手。

## 🚀 開發環境設置

本專案使用 React + Vite 開發。請確保您的電腦已安裝 Node.js (建議 v16+)。

1. **Fork 並 Clone 專案**
   ```bash
   git clone https://github.com/YOUR_USERNAME/proims-app.git
   cd proims-app
   ```

2. **安裝依賴套件**
   ```bash
   npm install
   ```

3. **啟動本地開發伺服器**
   ```bash
   npm run dev
   ```
   瀏覽器將自動開啟 `http://localhost:5173`。

## 🛠️ 提交變更 (Pull Request)

1. 請確保您的程式碼風格與專案一致。
2. 提交 Commit 時，請使用清晰的訊息，例如：
   - `feat: 新增匯出 PDF 功能`
   - `fix: 修正手機版面跑版問題`
   - `docs: 更新 README 文件`
3. 發送 Pull Request (PR) 到 `main` 分支。

## 🐛 回報問題 (Issues)
如果您發現 Bug 或有新功能建議，請至 Issues 頁面詳細描述問題重現步驟。
